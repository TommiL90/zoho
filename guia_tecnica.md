# Zoho - Desarrollador Web

## Guía para la tarea técnica de la entrevista

### Do's (Lo que debes hacer)

-   Convierte este archivo XD a HTML responsivo hasta 320px. Encuentra
    el enlace de XD/Figma en el archivo adjunto **design.txt**.
-   El ancho del diseño debe ser 100% del navegador, y la parte de
    contenido de la página debe tener un ancho de **1170px**.
-   Usa el archivo adjunto (**font.txt**) para la fuente **Zoho Puvi**
    en tu archivo CSS.
-   Crea e implementa una **imagen sprite** para todos los íconos y
    logos presentes en el diseño.
-   Usa la librería **jQuery** si es necesario.

### Don'ts (Lo que NO debes hacer)

-   Evita usar cualquier framework como **Bootstrap, Grid**, etc., tanto
    para CSS como para HTML.
-   Evita usar **plugins de JavaScript**.

### Orden de trabajo recomendado

1.  HTML limpio\
2.  Responsividad\
3.  Animaciones JS / CSS
